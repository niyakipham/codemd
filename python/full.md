### 3.3 Set (T·∫≠p h·ª£p) - T·∫≠p H·ª£p Kh√¥ng C√≥ Th·ª© T·ª± v√† Duy Nh·∫•t

Set l√† t·∫≠p h·ª£p c√°c ph·∫ßn t·ª≠ *kh√¥ng c√≥ th·ª© t·ª±*, ƒë∆∞·ª£c ƒë·∫∑t trong d·∫•u ngo·∫∑c nh·ªçn `{}` (ho·∫∑c d√πng h√†m `set()`) v√† m·ªói ph·∫ßn t·ª≠ trong Set *ch·ªâ xu·∫•t hi·ªán m·ªôt l·∫ßn* (duy nh·∫•t - unique). Set h·ªó tr·ª£ c√°c ph√©p to√°n t·∫≠p h·ª£p (h·ª£p, giao, hi·ªáu...).

### 9.3 Decorators - Th√™m Ch·ª©c NƒÉng Cho H√†m Kh√°c

Decorator l√† m·ªôt pattern (m·∫´u thi·∫øt k·∫ø) trong Python cho ph√©p b·∫°n d·ªÖ d√†ng th√™m ch·ª©c nƒÉng (nh∆∞ ghi log, ƒëo th·ªùi gian ch·∫°y, ki·ªÉm tra quy·ªÅn truy c·∫≠p...) v√†o m·ªôt h√†m ho·∫∑c ph∆∞∆°ng th·ª©c hi·ªán c√≥ m√† kh√¥ng c·∫ßn thay ƒë·ªïi c·∫•u tr√∫c c·ªßa h√†m/ph∆∞∆°ng th·ª©c ƒë√≥. N√≥ ƒë∆∞·ª£c bi·ªÉu di·ªÖn b·∫±ng c√∫ ph√°p `@ten_decorator` ngay tr∆∞·ªõc ƒë·ªãnh nghƒ©a h√†m.

M·ªôt decorator v·ªÅ c∆° b·∫£n l√† m·ªôt h√†m nh·∫≠n *m·ªôt h√†m kh√°c* l√†m ƒë·ªëi s·ªë, th·ª±c hi·ªán m·ªôt s·ªë c√¥ng vi·ªác b·ªï sung, v√† tr·∫£ v·ªÅ *m·ªôt h√†m m·ªõi* (th∆∞·ªùng l√† h√†m g·ªëc ƒë√£ ƒë∆∞·ª£c "g√≥i" l·∫°i).

```python
# V√≠ d·ª• Decorator ƒë∆°n gi·∫£n (ƒëo th·ªùi gian ch·∫°y)
import time

def do_thoi_gian_chay(func):
    def wrapper(*args, **kwargs): # *args, **kwargs cho ph√©p wrapper nh·∫≠n b·∫•t k·ª≥ s·ªë l∆∞·ª£ng tham s·ªë v·ªã tr√≠/keyword n√†o
        bat_dau = time.time()
        ket_qua = func(*args, **kwargs) # G·ªçi h√†m g·ªëc
        ket_thuc = time.time()
        print(f"H√†m '{func.__name__}' ch·∫°y m·∫•t {ket_thuc - bat_dau:.4f} gi√¢y.")
        return ket_qua # Tr·∫£ v·ªÅ k·∫øt qu·∫£ c·ªßa h√†m g·ªëc
    return wrapper # Decorator tr·∫£ v·ªÅ h√†m wrapper m·ªõi

# √Åp d·ª•ng decorator
@do_thoi_gian_chay

    print("K·∫øt th√∫c h√†m.")

@do_thoi_gian_chay

    for i in range(n):
        tong += i
    return tong

ket_qua_tong = ham_tong_lon(1000000)

```
Decorator l√† m·ªôt ch·ªß ƒë·ªÅ m·∫°nh m·∫Ω v√† ƒë√≤i h·ªèi hi·ªÉu bi·∫øt s√¢u h∆°n v·ªÅ h√†m b·∫≠c nh·∫•t (first-class functions) trong Python.

### 9.4 Generators v√† Iterators - X·ª≠ L√Ω D·ªØ Li·ªáu L·ªõn Hi·ªáu Qu·∫£ B·ªô Nh·ªõ

*   **Iterator:** M·ªôt ƒë·ªëi t∆∞·ª£ng c√≥ th·ªÉ "l·∫∑p" qua c√°c ph·∫ßn t·ª≠ c·ªßa m·ªôt t·∫≠p h·ª£p (nh∆∞ list, string, tuple...). Iterator c√≥ ph∆∞∆°ng th·ª©c `__iter__()` (tr·∫£ v·ªÅ ch√≠nh n√≥) v√† `__next__()` (tr·∫£ v·ªÅ ph·∫ßn t·ª≠ k·∫ø ti·∫øp, n√©m `StopIteration` khi h·∫øt ph·∫ßn t·ª≠). C√°c c·∫•u tr√∫c l·∫∑p nh∆∞ v√≤ng `for` trong Python l√†m vi·ªác ng·∫ßm v·ªõi iterator.
*   **Generator:** L√† m·ªôt c√°ch *d·ªÖ d√†ng* ƒë·ªÉ t·∫°o ra Iterator. Generator l√† c√°c h√†m *b√¨nh th∆∞·ªùng* nh∆∞ng thay v√¨ d√πng `return` ƒë·ªÉ tr·∫£ v·ªÅ gi√° tr·ªã m·ªôt l·∫ßn v√† k·∫øt th√∫c, ch√∫ng d√πng t·ª´ kh√≥a `yield` ƒë·ªÉ "sinh ra" (yield) m·ªôt gi√° tr·ªã m·ªói l·∫ßn g·ªçi, t·∫°m d·ª´ng tr·∫°ng th√°i h√†m v√† c√≥ th·ªÉ ti·∫øp t·ª•c ch·∫°y t·ª´ ƒë√≥ ·ªü l·∫ßn g·ªçi k·∫ø ti·∫øp. Generator ƒë·∫∑c bi·ªát h·ªØu √≠ch khi l√†m vi·ªác v·ªõi c√°c t·∫≠p d·ªØ li·ªáu *l·ªõn*, v√¨ ch√∫ng ch·ªâ t·∫°o ra v√† gi·ªØ trong b·ªô nh·ªõ *m·ªôt ph·∫ßn t·ª≠* t·∫°i m·ªôt th·ªùi ƒëi·ªÉm, thay v√¨ t·∫°o to√†n b·ªô t·∫≠p h·ª£p c√πng l√∫c.

```python

def dem_den(n):

        print(f"Sau yield {i}")

# ... c·ª© ti·∫øp t·ª•c

# Th∆∞·ªùng d√πng generator v·ªõi v√≤ng l·∫∑p for (v√≤ng for t·ª± ƒë·ªông g·ªçi next() v√† b·∫Øt StopIteration)

        a, b = b, a + b

# L·∫•y v√†i s·ªë Fibonacci ƒë·∫ßu ti√™n

```

### 9.5 Context Managers - T·ª± ƒê·ªông Qu·∫£n L√Ω T√†i Nguy√™n (`with` Statement)

B·∫°n ƒë√£ th·∫•y `with open(...)` ·ªü Ch∆∞∆°ng 6. C√∫ ph√°p `with` ƒë∆∞·ª£c s·ª≠ d·ª•ng v·ªõi c√°c ƒë·ªëi t∆∞·ª£ng g·ªçi l√† **Context Managers**. Context Manager ƒë·∫£m b·∫£o r·∫±ng m·ªôt t√†i nguy√™n s·∫Ω ƒë∆∞·ª£c qu·∫£n l√Ω (v√≠ d·ª•: setup tr∆∞·ªõc khi d√πng v√† cleanup sau khi d√πng), ngay c·∫£ khi c√≥ l·ªói x·∫£y ra.
